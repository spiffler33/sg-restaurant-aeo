# Sabai Fine Thai on the Bay — Web Presence Audit

**Date:** 2026-03-10
**Purpose:** Diagnose why search-augmented LLMs inconsistently surface Sabai Fine Thai, and identify actionable gaps.
**Context:** Sabai received zero mentions across 1,690 main-dataset queries. A targeted 160-query probe found 45% detection — but almost entirely in Tier 2-4 (location/attribute/name) queries, not generic "best Thai in Singapore."

## 1. Website Audit — sabaifinethai.com.sg

- Missing: name, servesCuisine, address, priceRange, menu, review, aggregateRating, geo coordinates

- **Meta title:** "Sabai | Fine Thai Dining By The Bay" — mentions "Fine Thai" and "Dining" but omits "Singapore" and "restaurant"
- **Meta description:** Not detected in crawled HTML — likely empty or auto-generated by WordPress

## 5. Diagnosis: Why Search ON Is Inconsistent

The probe found search ON sometimes helps (Gemini) but sometimes hurts (Perplexity) or has no effect (GPT-4o). Here's why:

### What search-augmented models actually see when they look up Sabai:

1. **TripAdvisor** (often rank #1 for the query) — 4.0 rating, 286 reviews. Moderate but not standout.
2. **OpenTable** — 4.8 but only 5 reviews. Too thin to be authoritative.
3. **Burpple** — 19 reviews. Negligible.
4. **Official website** — Homepage has no text. Menu is PDFs. No schema. No meta description.
5. **No recent blog coverage** to provide rich, keyword-dense descriptions.

When a model searches "best Thai restaurant Singapore" with web search enabled:
- It finds listicles dominated by Un-Yang-Kor-Dai, Nakhon Kitchen, Soi 47, Sarai
- It finds TripAdvisor/Yelp pages where Sabai doesn't rank high enough to appear in the first page of results
- It finds the official website but gets almost no useful text from it

**The inconsistency comes from which search results the model happens to fetch.** If it hits BestLah.sg (where Sabai is #1), it surfaces Sabai. If it hits any other listicle, HungryHub, or TripAdvisor's "10 Best," Sabai is absent.

### Gemini's advantage
Gemini uses Google Search grounding and is more likely to pull Google Business Profile data (where Sabai has 4.3 stars and the address shows "Customs House, Marina Bay"). This explains why Gemini has the highest detection rate (24/40) — it's reading structured Google data, not just web pages.

### Perplexity's disadvantage with search ON
Perplexity with `search_recency_filter="month"` narrows to recent content. Sabai has almost no recent blog or press coverage (2025-2026), so the filter actively excludes the few mentions that exist. This explains why Perplexity search ON (14 detections) is weaker than expected.

## 6. Prioritized Recommendations

### Immediate (this week — affects search ON within days)

**I-1. Add schema.org/Restaurant JSON-LD to the homepage.**
- Install a WordPress schema plugin (RankMath or Yoast SEO Premium) and configure:
  - `@type: Restaurant`
  - `name: "Sabai Fine Thai on the Bay"`
  - `servesCuisine: "Thai"`
  - `address` with `Marina Bay`, `Customs House`, `Collyer Quay`
  - `priceRange: "$$$"`
  - `geo` coordinates (1.2841° N, 103.8527° E)
  - `aggregateRating` from Google (4.3, 347 reviews)
  - `openingHours`
  - `menu` URL
  - `keywords: ["Thai fine dining", "waterfront restaurant", "Marina Bay", "bay view", "royal Thai cuisine"]`
- **Probe gap addressed:** Tier 1 visibility. Schema is what Gemini's grounding reads. No competitor has this — first mover advantage.

**I-2. Fix the meta title and add a meta description.**
- Current: `Sabai | Fine Thai Dining By The Bay`
- Proposed: `Sabai Fine Thai on the Bay — Thai Fine Dining Restaurant, Marina Bay Singapore`
- Meta description: `Authentic Thai fine dining at Customs House, Marina Bay. Chef Samlee, former cook to Thai royalty, serves royal Thai cuisine with waterfront views. Lunch & dinner Mon-Sat. Reservations available.`
- **Probe gap addressed:** Every search ON query. This is the first thing crawlers and LLM search tools see.

**I-3. Add text content to the homepage.**
- Below the hero images, add a brief text block (3-4 sentences):
  - "Sabai Fine Thai on the Bay is a Thai fine dining restaurant at Customs House, 70 Collyer Quay, overlooking Marina Bay. Chef Samlee, who trained cooking for Thai royalty, serves authentic royal Thai cuisine using the freshest ingredients sourced from Thailand. Open for lunch and dinner Monday to Saturday. Reservations via Quandoo, Chope, or OpenTable."
- **Probe gap addressed:** Search ON consistency. When LLM search tools fetch the homepage, they currently see nothing useful. This gives them a dense, keyword-rich description.

**I-4. Verify SSL certificate is valid.**
- Check `https://www.sabaifinethai.com.sg` in a browser. If "Not Secure" warning shows, renew the certificate through the hosting provider (Bluehost).
- **Probe gap addressed:** General credibility signal. "Not Secure" sites get deprioritized by both search engines and cautious LLM tools.

### Short-Term (1-2 weeks — affects search ON within a month)

**S-1. Create an HTML menu page.**
- Convert the PDF menu to crawlable HTML text on `/menu/`
- List dish categories (Starters, Soups, Curries, Mains, Desserts) with dish names and brief descriptions
- Include prices
- Add `schema.org/Menu` and `MenuItem` markup
- Keep PDF as a secondary download link
- **Probe gap addressed:** Tier 3 attribute queries. When someone asks "Thai restaurant with great curries in Singapore" or "Panang curry Singapore," the menu text makes Sabai discoverable.

**S-2. Create a proper "About" page with location keywords.**
- Rewrite `/about-us/` to include:
  - "Located at Customs House overlooking Marina Bay" in the first paragraph
  - "Near Raffles Place MRT and Downtown MRT"
  - "Waterfront Thai dining with views of Marina Bay Sands"
  - "Serving Singapore since 2004"
  - Chef Samlee's royal Thai credentials prominently
- **Probe gap addressed:** Tier 2 location queries. "Marina Bay" is the #1 location keyword that drives discovery. It must appear on the website.

**S-3. Update the copyright and food menu date.**
- Footer: "© 2004-2026 Sabai Fine Thai on the Bay"
- Upload current menu PDF (the food menu is dated Dec 2023)
- **Probe gap addressed:** Signals that the restaurant is active. Stale content reduces crawl priority.

**S-4. Fix non-SEO-friendly URLs.**
- Change `/299-2/` to `/contact/`
- Change `/?page_id=276` to `/delivery/`
- Set up 301 redirects from old URLs
- **Probe gap addressed:** Minor but contributes to overall site authority.

**S-5. Submit an XML sitemap to Google Search Console.**
- Install Yoast SEO (if not already) → generates sitemap automatically
- Submit at `https://search.google.com/search-console/`
- **Probe gap addressed:** Ensures Google crawls all pages efficiently, especially the new HTML menu page.

**S-6. Solicit Google reviews.**
- Current: 347 reviews (4.3 stars). Target: 500+ reviews within 3 months
- Place a QR code at table tents linking to the Google review page
- Encourage review keywords: "bay view," "Marina Bay," "fine dining," "best Thai," "waterfront"
- **Probe gap addressed:** Tier 1 generic queries. Review volume is the single strongest predictor of LLM recommendation. Sarai has 3.2x more, Sawadee has 8x more.

### Medium-Term (ongoing — affects future parametric training)

**M-1. Pitch food bloggers for coverage.**
- Target: DanielFoodDiary, ieatishootipost, MissTamChiak, SetHLui, EatBook, TheRantingPanda
- These are the blogs that LLMs read during training. Sarai has coverage from 4+ of these. Sabai has near-zero.
- Angle: "The 20-year-old Thai fine dining restaurant you've never heard of — Chef Samlee cooked for Thai royalty"
- **Probe gap addressed:** Parametric knowledge (search OFF). When LLMs are trained on web corpora, blog posts become parametric knowledge. Sabai has almost no representation in the training data, which is why search OFF detection is so low.

**M-2. Get listed on HungryGoWhere.**
- Sabai was not found on HungryGoWhere, one of Singapore's biggest food platforms
- HungryGoWhere listings are frequently cited in LLM responses
- **Probe gap addressed:** Platform breadth. More listings = more signals for search-augmented models.

**M-3. Seek "best Thai" listicle placement.**
- Approach HungryHub, BestPicks.sg, EatBook, TheSmartLocal for inclusion in their Thai restaurant roundups
- The BestLah.sg placement (ranked #1) is excellent but it's only one listicle
- **Probe gap addressed:** Tier 1 generic queries. When LLMs search "best Thai restaurant Singapore," they read these listicles. Appearing in 3-4 of them instead of 1 dramatically improves generic detection.

**M-4. Differentiate from Sarai in name.**
- The name confusion is real: 17 "mentions both" instances in 160 queries
- Consider consistent branding as "Sabai Fine Thai on the Bay" (never just "Sabai Fine Thai")
- Ensure all platforms use the full name with "on the Bay"
- Emphasize "Customs House, Marina Bay" in all descriptions to differentiate from Sarai's "Tanglin Mall"
- **Probe gap addressed:** Name confusion. Models that know both Sabai and Sarai sometimes conflate them or recommend Sarai when the user is near Marina Bay.

**M-5. Build a "bay view" / "waterfront" content signal.**
- The website never says "bay view" or "waterfront" in text
- Neither do most reviews
- Yet "Thai restaurant with a bay view in Singapore" is Sabai's #1 breakout prompt (detected at #1 on 5/8 model×search combos)
- Encourage staff to mention views when prompting reviews
- Add a dedicated "Waterfront Dining" section to the website with photos of the bay view
- **Probe gap addressed:** Tier 3 attribute queries. The bay view is Sabai's strongest differentiator, but it's invisible in text data.
